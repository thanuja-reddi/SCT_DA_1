# SuperStore Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project presents an interactive Sales Analysis Dashboard created using the SuperStore Sales Dataset in Microsoft Excel. The dataset was cleaned, analyzed, and visualized to uncover key business insights related to sales performance, profitability, customer segments, and regional trends.

The dashboard enables users to explore data dynamically through slicers and visualizations, helping stakeholders make informed business decisions.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning & Transformation

## 📂 Dataset Information
The dataset contains sales transaction details, including:
- Order Date
- Sales
- Profit
- Quantity
- Discount
- Category
- Segment
- Region
- Order ID and other order-related attributes

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)
The dashboard highlights important business metrics:
- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Quantity Sold
- 🏷️ Total Discount
- 🛒 Total Orders

### 📈 Visualizations

#### 1. Region-wise Sales Distribution
**Chart Type:** Pie Chart

Shows the contribution of each region to overall sales.

#### 2. Profit by Category
**Chart Type:** Column Chart

Compares profit generated across product categories.

#### 3. Monthly Profit Analysis
**Chart Type:** Bar Chart

Displays profit trends over different months.

#### 4. Sales by Category
**Chart Type:** Pie Chart

Illustrates the share of total sales contributed by each category.

#### 5. Monthly Sales Trend
**Chart Type:** Line Chart

Tracks sales growth and seasonal patterns over time.

## 🎛️ Interactive Filters
The dashboard includes slicers that allow users to analyze data dynamically by:
- Category
- Region
- Segment
- Year

These filters make the dashboard highly interactive and user-friendly.

## 🧹 Data Cleaning & Preparation
Before creating the dashboard, the dataset underwent several preprocessing steps:
- Removed duplicate records
- Handled missing values
- Standardized data formats
- Verified data consistency
- Converted and formatted date fields correctly

## 🔍 Key Insights
- Technology products generated the highest profit.
- The West region contributed the largest share of sales.
- Sales showed noticeable growth during year-end months.
- Office Supplies maintained consistent sales performance throughout the period.
